Key Design Decisions

  • UTC storage — all times stored in UTC, converted to IST (UTC+5:30) at display
  • Haversine on server — geofence check runs server-side using branch lat/lng + radius
  • One attendance record per day — UNIQUE(employee_id, date) prevents duplicates
  • Leave balances per year — supports carry-forward from previous year
  • Sensitive data encrypted — passwords bcrypt hashed, PAN/Aadhar stored with AES encryption

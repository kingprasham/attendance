All Dashboard Pages

📈 Dashboard Home

Daily overview with key metrics

  • Today's attendance summary
  • Branch-wise breakdown
  • Pending leave requests count
  • Recent activity feed

🏭 Branch Management

Create and configure branches

  • Add branch with Google Maps pin
  • Set geofence radius (meters)
  • View employee count per branch
  • Activate / deactivate branches

👥 Employee Management

Full employee CRUD

  • Create with all details + salary
  • Assign username + password
  • Assign to branch
  • Reset device binding
  • Activate / deactivate

⏱ Daily Attendance Report

Real-time daily tracking

  • Filter by date + branch
  • Present / absent / late / leave status
  • Clock in/out times + hours worked
  • Export as PDF / Excel

✈️ Leave Requests

Approve or reject leave applications

  • Pending requests queue
  • View employee balance before approving
  • Add admin remarks
  • Push notification on action

💰 Payroll Generation

Monthly salary processing

  • Select branch + month
  • Auto-calculate from attendance
  • Review before generating
  • Generate salary slips
  • Notify employees

📆 Leave Policies

Configure per-branch leave rules

  • Set annual quota per leave type
  • Enable/disable carry-forward
  • Max carry-forward days
  • Copy policy across branches

📅 Holiday Calendar

Manage company holidays

  • Add/remove holidays per branch
  • Mark as optional or mandatory
  • Calendar year view

⚙️ Admin Settings

Account and system config

  • Change password
  • API base URL config
  • Working hours definition
  • Late threshold (e.g., after 9:30 AM)

Dashboard Tech Stack

HTML5 + CSS3 (responsive)
Vanilla JavaScript (ES6+)
Bootstrap 5 (UI framework)
Chart.js (attendance graphs)
DataTables (sortable tables)
Google Maps JS API (branch pin)
PHP sessions (admin auth)
FPDF / DomPDF (PDF export)
